# Contributing to the Curvachip Website

Thank you for your interest in contributing to the official Curvachip website.

This repository contains the production website for Curvachip LLC. While community contributions are appreciated, this project is proprietary and is not open source.

## Before You Contribute

Before investing significant time in a feature or large change, please open an Issue to discuss it first. This helps ensure that proposed changes align with the project's goals and avoids duplicate work.

## Ways to Contribute

Contributions may include:

- Bug fixes
- Accessibility improvements
- Performance optimizations
- Documentation improvements
- Typographical corrections
- Code quality improvements
- Small UI enhancements
- Security improvements

Large design changes, branding updates, or architectural changes generally require prior approval.

## Development

### Prerequisites

- Node.js (latest LTS recommended)
- npm
- Git

### Getting Started

```bash
git clone https://github.com/<your-org>/<repo>.git
cd <repo>
npm install
npm run dev
```

## Coding Guidelines

Please:

- Keep pull requests focused on a single change.
- Follow the existing code style.
- Use descriptive variable and function names.
- Avoid unnecessary dependencies.
- Test your changes before submitting.
- Update documentation when applicable.

## Pull Requests

When submitting a pull request:

- Provide a clear description of the change.
- Explain why the change is needed.
- Reference related Issues when applicable.
- Keep commits clean and descriptive.

Pull requests may be edited, modified, or declined at the discretion of Curvachip LLC.
